# MangaBinder

[English README](README.md)

MangaBinder は、自炊漫画画像をスマホで快適に読むための「製本工程」に特化した Windows デスクトップアプリです。

一般的な漫画サーバやストリーミング型ライブラリ管理ツールとは異なり、MangaBinder は「読む前の準備工程」を重視しています。

主な目的：

- 漫画アーカイブの整理
- 巻構成の再構築
- スマホ向け ZIP 出力
- 見開き分割などの前処理
- 問題のあるアーカイブ構造の確認
- 大規模な個人漫画ライブラリの管理

実際の自炊運用・長期保存運用を前提として開発しています。

## 設計思想

MangaBinder は、汎用的なメディアサーバ管理ではなく、実際の個人運用を重視して設計しています。

目的は「漫画を配信する事」ではなく、

1. 漫画アーカイブを集める
2. バラバラな素材を整理する
3. 読みやすい状態に加工する
4. 長期的に管理する

という流れを改善する事です。

特に、日本語漫画コレクションや日本語ファイル名文化を強く意識しています。

## 主な機能

### ライブラリ管理

サムネイル、作者、メタデータ、あらすじなどを表示しながら漫画ライブラリを管理できます。

### アーカイブ構造確認

製本前に複雑なアーカイブ構造を確認できます。

例：

- Nested Archive
- 不正フォルダ構造
- 巻構成問題
- 非対応ファイル

### 見開き分割ワークフロー

見開きページを選択し、分割位置を調整できます。

### メタデータ補助

Google Books の情報を利用し、作品検索性や発見性を向上できます。

### バックグラウンド処理

サムネイル生成などの重い処理はバックグラウンドジョブで実行します。
